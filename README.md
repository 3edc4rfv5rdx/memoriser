## Memorizer
Android memory assistant:

## Features
- English, Ukrainian, Russian localization (languages can be easily added).
- Allows you to record events, their description, priority,
  tagging, add photos, dates, set reminder times and delete
  after triggering, set annual events (such as birthdays, stored
  in a virtual folder).
- Filters are available for all fields and tags.
- You can save and restore the database in DB (Sqlite) and CSV files.
- Built-in help system for all interface elements.
- There is a PIN-protected area to hide from prying eyes (unencrypted,
  easy base64 conversion) - accessed by tapping the Memorizer header four times.
- Color themes.
